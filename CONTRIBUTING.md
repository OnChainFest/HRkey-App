# Contributing

## Scope
- OPEN paths: contributions welcome.
- CLOSED paths: maintainer approval required.
- REVIEW paths: treat as CLOSED unless explicitly approved.

## Restrictions
- Do NOT include ML datasets or model weights.
- Do NOT include HRScore/scoring formulas, weights, or pricing logic.
- Do NOT include secrets or credentials.

## Rights
By submitting a contribution, you certify that you have the right to submit the code and that it does not infringe third-party rights.
